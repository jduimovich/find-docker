App Studio Demo Application

Use Red Hat Trusted Application Pipeline to Build your Application

This application is a standard Node.js HTTP application packaged into a single container.

Demo-palooza! Run this and more using a script launcher found here https://github.com/jduimovich/appstudio-e2e-demos

Try

  1. Bootstrap your cluster from https://github.com/redhat-appstudio/infra-deployments 
     ./hack/bootstrap.sh
  2. Create an Application and a Component 
    3.     apiVersion: appstudio.redhat.com/v1alpha1
    kind: Application
    metadata:
      name: single-container-app
    spec:
      description: "single-container-app built in Stone Soup"
      displayName: single-container-app 
    
        apiVersion: appstudio.redhat.com/v1alpha1
    kind: Component
    metadata:
      name: single-container-app
    spec:
      componentName: single-container-app
      application: single-container-app
      containerImage: quay.io/jduimovich0/single-container-app
      source:
        git:
          url: https://github.com/jduimovich/single-container-app
          devfileUrl: https://raw.githubusercontent.com/jduimovich/appstudio-e2e-demos/main/demos/single-container-app/devfiles/devfile.yaml
  3. Stone soup will build and deploy your application. In the manage stone soup environment use the UI to browse to your running application. If are using CLI only to access, use `oc get routes` to find the URL to your application.
